# Product

## Current feature set
- AI-agent-first PR analysis workflow
- canonical GitHub review summary contract
- Slack/Discord-compatible notification payloads
- repository memory scaffold
- analytics dashboard
- replayable test harness

## User journeys
### Repository admin
- connect GitHub
- install GitHub App
- route notifications
- inspect activity and analytics

### Reviewer
- receive realtime triage summary
- jump directly to the PR
- use attention level and reviewer focus to prioritize work

### Team lead
- inspect risky PRs quickly
- watch confidence and attention distribution over time

## Non-goals in this repo
- BYOK UI
- public self-serve billing
- auto-drafting PR descriptions
- autonomous review actions
